# game_structure_A.md
# 《트릭컬: 아무도 안 했습니다》 — A안 전체 구조 설계
# 단간론파형 폐쇄 군상극 · 비살상("기절") 버전

> 이 문서는 게임 전체의 뼈대(전제·규칙·진행자·챕터 흐름)를 정한다.
> 개별 사건(case01 등)은 이 골격 위에 얹는다. 첫 개발 목표는 여전히 **서장 + 제1막 1개 완성**.
> 표기: **[제안]** 추천안 / **[갈림길]** 네가 골라야 할 지점.

---

## 0. 로그라인

> 교주와 사도들이 어떤 "이야기" 속에 통째로 갇힌다. 빠져나가려면, 매 막(幕)마다 벌어지는 **기절 사건**의 범인을 교단 심문회에서 밝혀내야 한다. 그러나 이 폐쇄를 뒤에서 짜고 있는 자가 따로 있다.

---

## 1. 세계관 전제 — 왜 갇혔나

### [제안] 메인 안: "이야기 속 무대"
마카샤(이야기를 짓는 자 / 차원 오두막)와 셰럼(자기가 쓴 책 속 인물을 실체화하는 사관)의 능력이 얽혀, 교단 사도들이 **하나의 책/이야기 세계 안으로 빨려 들어간다.** 그 안은 닫힌 무대이고, 바깥으로 나가려면 "이야기"가 요구하는 규칙(=기절 게임)을 끝까지 클리어해야 한다.

이 전제의 장점:
- **자기 정당화:** "왜 갇혔고 왜 추리 게임을 강요당하나"가 캐릭터 능력으로 자연히 설명됨.
- **죽은 자·변장 허용:** 책 속 무대라 우로스(책에서 실체화되는 폭군)도 등장 가능하고, 스피키의 변장 트릭, "기절=책장에 봉인" 같은 비살상 연출이 전부 설정으로 말이 됨.
- **교주 캐논 활용:** 교주는 원작에서 돌 맞고 기억을 잃는 인물. 게임은 **교주가 기절했다 깨어나 "왜 다들 여기 갇혀 있지?"** 로 시작 → 기억·정보가 비어 있는 상태라 플레이어와 동기화됨.

### [갈림길] 대체 안: "교단 봉쇄"
뮤트의 정보 결계 + 죠안의 난 재발로 교단 본부가 물리적으로 봉쇄됨. 더 직설적이지만, 책 안만큼 연출 자유도(소환·변장·비살상)가 안 나옴. → 메인 안 권장.

## 2. 핵심 규칙 — "기절 게임"

단간론파 루프(폐쇄 → 사건 → 추리 → 재판 → 탈락)를 그대로, 살상만 치환.

### 한 막의 진행
```
막 시작(일상·관계) → 기절 사건 발생 → 현장 조사 → 단톡방·게시판·기록 조사
→ 개별 심문 → 교단 심문회(논점 격파) → 사건 재구성 → 범인 지목 → 처분 → 후일담
```

### 사건 = "기절"
- 누군가 **꿀밤·마법·함정·투구꽃(약물)·청소기 폭주(스피키)** 등으로 사도 1명을 **기절/리타이어**시킨다.
- 트릭컬에선 강한 일격 한 방에 몇 시간 기절하는 묘사가 표준이라 세계관 정합성 OK. 피 없음, 죽음 없음.

### 판정 & 페널티
- 심문회에서 **범인을 맞히면** → 그 막 클리어, 범인 처분.
- **틀리면** → 교주 멘탈 감소(체력). 멘탈 0 = 막 실패(배드엔딩 분기).
- 난이도 모드로 멘탈 압박 조절(스토리/기본/하드).

### 처분 = 비살상 탈락 [제안]
범인으로 지목된 자는 **"책장에 봉인"**(페이지가 닫히며 무대에서 퇴장). 단간론파의 처형 자리지만 죽음이 아님.
- 코믹 치환 후보: 책장 봉인 / "주말농장으로 추방"(트릭컬 밈) / 무저갱(죠안이 주장하던 엄벌) 패러디.
- **복귀 가능:** 비살상이라 봉인된 캐릭터를 종막에서 다시 펼쳐 복귀시킬 수 있음. 기절한 피해자도 다음 막에 깨어나 재합류 가능 → **로스터가 단간론파처럼 가혹하게 줄지 않음**(팬게임에 유리).

## 3. 진행자 & 흑막 (모노쿠마 구조 분리)

단간론파의 "진행자(모노쿠마)"와 "흑막(마스터마인드)"을 분리해서 배치.

| 층위 | 담당 [제안] | 역할 |
|---|---|---|
| 무대 주인 | 마카샤 / 셰럼 | 책·이야기 세계 자체를 만든/관리하는 존재. 본인들도 갇힘 or 공범인지는 떡밥 |
| **진행자**(모노쿠마 役) | **죠안** | 룰 선포·도발 담당. "이 중에 가짜(범인)가 있다, 찾아내라"를 세계관 안에서 정당화. 원작에서 교주에게 마피아 게임을 시전한 전적이 있어 적격 |
| **흑막**(마스터마인드) | **뮤트** [확정 필요] | 이 폐쇄를 막후에서 짜고 이용하는 진짜 배후. 이름 Mute = "pull the strings(막후 조종)". 종막에서 정체 공개 |

> [갈림길] 흑막을 뮤트로 확정할지, 또는 마카샤/셰럼/R-41 아멜리아로 트위스트를 줄지는 종막 설계 때 최종 결정. 지금은 뮤트를 잠정 흑막으로 두고 떡밥만 깔아둠.

## 4. 챕터(막) 매크로 흐름

전체는 **서장 + 제1막~제5막 + 종막** 정도를 최종 목표로(단간론파 6장 구조 참고). 막이 진행될수록 사건이 무거워지고 흑막 떡밥이 쌓임.

```
서장      : 갇힘. 규칙 선포(죠안). 교주 각성. 첫 일상.
제1막     : 가벼운 사건. 튜토리얼격. 범인=오해/선의형. ("아무도 안 했습니다" 톤 확립)
제2~3막   : 트릭 본격화. 스피키 변장·기록 조작 등 도입.
제4막     : 무게감 상승. 우로스(후반 보스)·디아나 트라우마 등 진지 라인.
제5막     : 흑막 윤곽. 뮤트 떡밥 회수 직전.
종막      : 흑막(뮤트) 대치. 봉인된 동료 복귀. 탈출.
```

- **첫 개발 = 서장 + 제1막**만 완성하면 "게임 만들었다"고 말할 수 있는 단위.
- 이후 막은 데이터(사건/증거/심문) 추가 방식으로 확장.

### 막마다 등장 인원
14명을 매 막 전원 핵심으로 굴리지 않음. **한 막에 6~8명 집중 등장 + 나머지는 배경/단톡방.** (단간론파도 매 재판마다 전원이 핵심은 아님)

## 6. 교주(플레이어) & 멘탈

- 1인칭 무얼굴. 탐정 = 플레이어 본인.
- **멘탈 게이지 = 체력.** 오답·무리한 추궁 시 감소, 0이면 막 실패.
- 원작 능력 "독심술"을 보조 힌트 시스템으로 차용 가능(상대가 숨기는 게 있는지 정도만 감지 → 힌트 1단계).

---

## 7. 톤 가이드 — "아무도 안 했습니다"

- **매 막의 범인은 진짜 악인이 아니다.** 오해·선의·사소한 욕심(간식)·내기·흑막의 조종 등 코믹하거나 딱한 동기. 기절시킨 것도 사고이거나 오인인 경우가 많음.
- **진짜 악의는 흑막 한 곳으로 수렴.** 그마저도 트릭컬 특유의 하찮음을 섞어 너무 어둡지 않게.
- 우로스 같은 무거운 캐릭터는 **의도된 대비**로만 투입(후반 긴장 담당).
- 전체적으로 역전재판/단간론파의 골격 + 트릭컬의 귀엽고 나사 빠진 톤.
